Version 1.0

1.搭建

  • 最初在 win 上的时候是在cnfeat的指导下完成的(总的文档在此),但是在 mac 下安装 node.js 的时候貌似遇到权限问题,后来是通过这么解决:

    $sudo chown -R `whoami` ~/.npm
    $sudo chown -R `whoami` /usr/local/lib/node_modules

  • 主题改的是 jacman

    $ git clone https://github.com/wuchong/jacman.git themes/jacman

  • jacman的主题在3.0.0-rc.4的 Hexo 版本中有 bug, 无法渲染 css 文件.因此要用这个主题模板的话还是退版本保平安(根据github 讨论)

    npm uninstall hexo -g
    npm install hexo@2.8.3 -g

  • 安装了两个插件:

    $ npm install hexo-generator-sitemap
    $ npm install hexo-generator-feed
